# Handwritten Digit Classification using ANN

This project uses an Artificial Neural Network (ANN) to classify handwritten digits using the MNIST dataset.

## Features
- Data preprocessing
- ANN model using TensorFlow/Keras
- Training and evaluation
- Prediction of handwritten digits

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## Dataset
MNIST dataset (handwritten digits 0–9)

## How to Run
Open the notebook in Google Colab and run all cells.
